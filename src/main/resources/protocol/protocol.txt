//响应内容 json 字段说明：
//| 字段    | 说明                                                         | 类型           |
//| ------- | ------------------------------------------------------------ | -------------- |
//| code    | 响应编码，见 [鉴权响应编码表](#鉴权响应编码表)               | 32位无符号整型 |
//| message | 描述                                                         | 字符串类型     |

//| object  | { <br>session: 鉴权代理会话ID，同请求内容的session字段；字符串类型 <br>expire: 鉴权结果失效时间点，UTC时间（1970年以来），单位秒；32位无符号整形 <br>} |                |
//**注意**：通过expire可以控制鉴权请求周期。
//| 编码  | 说明                   | 备注                     |
//| ----- | ---------------------- | ------------------------ |

//| 10002 | app token 校验失败     | app token 校验失败       |
